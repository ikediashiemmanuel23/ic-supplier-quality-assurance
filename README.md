# ABSTRACT
This project provides a comprehensive insight on suppliers' performance and materials' Quality Assurance (QA) highlighting how key metrics such as defect quantity and downtime hours impacts overall operations. The aim was to transform an industrial static data to create a Prescriptive Engine that justifies automated strategic actions and closes global performance gaps.

Materials & Methods: The report utilized Microsoft Excel for initial data structuring and Power BI as the primary analytical environment. The methodology followed a chronological progression:

Data Modeling: Creating relationships between tables and applying the 6 Robust Principles (including the MECE framework) to ensure mutually exclusive data categories.

Multilayered Analysis: Integrating Descriptive, Diagnostic, Predictive, and Prescriptive analysis types.

Advanced Visualization: Implementing Temporal Heatmaps and Location-Based Drill-Throughs to isolate root causes.

Strategic Logic: Developing custom DAX measures to automate recommendations for site-level training versus vendor audits.

The project successfully transitioned raw data into a strategic roadmap by utilizing advanced drill-throughs for a seamless transition between report layers, significantly increasing decision-making efficiency and reducing "Time-to-Insight."

It is recommended that this prescriptive logic be scaled across all supply chain tiers to maximize cost avoidance, operational precision and  provide understanding of material risk levels.
